Pop-Up box

(IOT Project)

The Popup Box is an Ambient display for your desk which tells you which country is being talked about (in this case USA and India) frequently in twitter during that time. In our case, if the word 'America' pops up in any of the tweet at that point of time the blue circle pops out and if the word 'India' pops up the red circle pops out. We used Particle Photon controller, couple of stepper motors, laser cutter and 3-D printer to built this project. We used Javascript and assembly language to pull live data from the Twitter API and control the photon controller respectively. This is a working prototype which is very easy to make and customizable..
