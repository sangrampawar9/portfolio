Wear-emin - A digital Theremin

About

Wear-emin, a compact, digital music synthesizer that is controlled through hand motions designed to replicate those normally used to operate a traditional Theremin as to best provide an experience similar to operating a classic Theremin as possible. The Wear-emin is paired with a web application that provides the user with an interface that helps users to learn how to use the device and become more proficient in using it by giving them visual feedback for their actions. We conducted a small, informal study to test the device with two users and see how using the web application affected the learning experience. We discuss the creation of the device and web application, along with our findings from the user study that we conducted and their implications.

Wear-emin

How Does it Work?

To build the Wear-emin we used a Particle Photon, which is a prototyping microcontroller used in creating Internet of Things devices, as its core. The Photon is used to govern the device’s operations as well as connect to an associated web application that may access and display the Wear-emin’s output via WiFi. The Photon has attached to it an Inertia Measurement Unit (IMU) shield which works easily to process gesture based input and provides for further connection to the device’s proximity sensor and speaker. The integrated accelerometer inside the IMU returns metrics detailing the user’s movement, whose values are in turn mapped to controlling the pitch of the sound to be produced by the Wear-emin. The proximity sensor takes in infrared light to gauge the distance between the user's thumb and the sensor’s lenses. The sensor translates this input into a binary; if a user’s thumb is out of the sensor’s line of sight the Wear-emin will pause all further sound until the user moves their thumb back front of the sensor, allowing users to control the pace of the beat at which they play. The last major component is the piezoelectric speaker, a simple buzzer from which all of the sounds produced by the Wear-emin are emitted.

Implementation

    Process

  • The web application was designed to provide visual feedback to the user about the notes being played based on the input the application gets from the proximity sensor via the Photon.
  • The Photon has a publish function which can be used to send data to the cloud. The web application then takes this data from the cloud to give its feedback to the user.
  • The web application contains an array of 8 blank rounded rectangles that represent each note in an octave.
  • the user plays the Wear-emin, whichever note the user is currently playing will be colored in. Previously, the boxes were arranged in a 3x2 grid, however we changed this to a vertical line of boxes to better mirror the physical motion of moving up and down that navigating the scale necessitates.
  • Alongside this, displayed the melody of “Happy Birthday” for the purpose of this study. The application displays the individual notes for the song which can be used as guidance as the user plays along.
