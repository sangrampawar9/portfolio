Gomasu- NGO Website Design

About the Project

GOMASU is a non-profit that strives to help out underprivileged children and children with various disabilities. They wanted a website that could reach out to people and involve them in the process. This website would show the work done by the organization.
Note: This project is in progress.

Process

I worked with Chandan to help design a website for GOMASU. We conducted user interviews to get an idea about the target audience, and we also looked at other nonprofit organizations to help us during the ideation process.
The wireframe of the website was then done using Balsamiq software. Lastly, we did the visual design of the website with Sketch and Adobe Experience Design software.

Target Audiences

After analyzing the users of other NGO websites and the user interviews that we conducted, we categorized users into three types- Volunteers, Donors and Casual visitors.

  • Volunteers would like to get involved and help the organization.
  • Donors would like to provide financial aid.
  • Casual visitors would like to learn more about GOMASU Development Foundation.

Wireframe Samples

We did not want users to be overloaded with information. We wanted to provide general information about the website in a simple manner. This led us to use a carousel that would display a couple of scenarios about the work done by the nonprofit when a visitor first opens the website.
We decided that the volunteer and donor section would not contain flashy graphics that could annoy the user. Instead, we used volunteer and donate buttons that are easy to perceive.

Challenges

  • It took time to understand the identity of the NGO.
  • Designing the website from the user’s (Volunteers and Donor) perspective; to provide just enough information to encourage the user to act upon it.
  • Making the website design fully responsive so it could be used on any device.
