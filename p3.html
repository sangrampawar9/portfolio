Voice Controlled Home Automation

Working

This project deals with the development of a ‘Voice Controlled Home Automation’ System.It involves the control of multiple household electrical devices such as lights, fans as well as other home appliances via voice commands. The aim of this project is to provide a convenient way for users to control their home appliances with minimum physical effort. This concept is especially useful for the disabled or infirm as it provides them a way to control their home appliances remotely through voice, thereby granting them greater independence and minimum efforts in their homes. It offers a comfortable environment for others.

HM 2007 Speech Recognition Circuit

How Does it Work?

A unit will be held in the user’s hands, which will consist of a wireless RF transmitter. The user will give the voice command in the transmitter, and this command will be transmitted to the receiver end, which is stationary near the switch board in the house. The output of the receiver will be connected to the input of the Voice Recognition Module. This module will then decode the voice and give outputs accordingly, where we have to first store commands in the module and train it, it is all explained below. This decoded output of the voice recognition module will be given to the microcontroller and accordingly, the microcontroller will know which relays to be operated on, whether to be switched ON or OFF. Thus, we are able to control electrical devices using voice command.

Schematic and Implementation

    Process

  • FM Transmitter: This block consists of a mic, from which voice input will be given, and this voice will be transmitted to the central unit, through the receiver.
  • FM Receiver: This block will receive the voice input given to the mic in the transmitter. Output of this block will be connected to the input of Voice Recognition Module.
  • Voice Recognition Module: The voice commands required to control the appliances will be stored initially. A lookup table will be created in the memory. This block will take the input from FM receiver and correctly recognize which command was spoken, and give an 8-bit output.
  • Microcontroller: 8-bit output from the Voice Recognition Module will be matched to the command stored in the microcontroller, such that the microcontroller will know which relay to drive on reception of a particular command. Accordingly, microcontroller will give an 8-bit serial command to the relay driver.
  • Relay Driver Circuit: 8-bit serial command from the microcontroller will be received and accordingly 8 outputs of relay driver IC will be controlled.
  • Load: Thus, driving of electrical appliances in homes using voice recognition technique is possible.
  • Power Supply: This unit will supply power to all units of the system for working.
